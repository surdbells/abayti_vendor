Verification code

The verification code input plugin is a lightweight JavaScript solution for handling multi-input verification codes. The data-toggle="verification-code" attribute initializes the plugin when the page loads. The data-length="n" attribute tells the plugin that there should be n input fields for the verification code. The plugin will automatically manage input behavior, including cursor movement and paste functionality, without requiring any extra configuration or event handling.

<div class="hstack gap-1" data-toggle="verification-code" data-length="6" style="max-width: 20rem">

Interactive table rows

By default, table rows cannot be links or buttons. To enable row-based navigation, a small JavaScript enhancement has been added to make entire rows clickable. However, this affects the behavior of interactive elements nested within these rows. To ensure usability, we've implemented a fix that allows normal interaction with the following elements: <a>, <button>, and <input>. Other interactive elements may not behave as expected within clickable rows.
